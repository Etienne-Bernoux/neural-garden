# Interface TUI

## Layout

| Zone | Position | Contenu |
|---|---|---|
| Jardin | Haut-gauche (60%) | Rendu Braille. Chaque espèce = couleur ANSI distincte. |
| Stats | Haut-droite (40%) | Génération, fitness, biodiversité, symbioses, couverture |
| Courbe | Bas-gauche (60%) | Fitness + nb espèces vivantes au fil des générations |
| Logs | Bas-droite (40%) | Naissances, morts, symbioses, invasions, saisons |

## Hotkeys

| Touche | Action |
|---|---|
| Espace | Pause / Reprendre |
| + / - | Vitesse |
| s | Sauvegarder population |
| r | Exporter replay JSON |
| q | Quitter |
| Tab | Cycle plantes |
| d | Détail cerveau |
| m | Toggle mycélium |
| e | Toggle exsudats |
